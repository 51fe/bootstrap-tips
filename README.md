> 本实例实现了 Bootstrap 模态框垂直居中。

## 教程

[Bootstrap 模态框垂直居中](https://riafan.com/vertical-center-bootstrap-modal.html)

## 打包

1. 安装nodejs: https://nodejs.org/en/
2. 打开命令行程序，切换到当前工程的根路径
3. 安装 grunt 及插件

```bash
npm install
```
4. 将 `app` 中的文件合并压缩后打包到 `dist` 文件夹中

```bash
npm run build
```

## 部署

最终分发 `dist` 文件夹的文件
