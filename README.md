> 本实例实现了自定义 Bootstrap 下拉框。

## 教程

[自定义 Bootstrap 下拉框](https://riafan.com/customize-bootstrap-select.html)

## 打包

1. 安装nodejs: https://nodejs.org/en/
2. 打开命令行程序，切换到当前工程的根路径
3. 安装 grunt 及插件

```bash
npm install
```
4. 将 `app` 中的文件合并压缩后打包到 `dist` 文件夹中

```bash
npm run build
```
## 部署

最终分发 `dist` 文件夹的文件
