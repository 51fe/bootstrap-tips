> 本实例实现了 bootstrap 模态框动态添加内容。

## 教程

[为 Bootstrap 模态框注入内容](https://riafan.com/bootstrap-modal-injection.html)

## 打包

1. 安装nodejs: https://nodejs.org/en/
2. 打开命令行程序，切换到当前工程的根路径
3. 安装 grunt 及插件

```bash
npm install
```
4. 将app中的文件合并压缩后打包到dist文件夹中

```bash
npm run build
```

## 部署

最终分发 `dist` 文件夹的文件
